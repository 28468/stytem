#考试管理系统
<h2>1.项目简介</h2>

````
考试管理系统,主要为学生考试而创建的

````
<h2>2.技术栈</h2>

````
"dva","hooks","router","scss","redux",

````
<h2>3.安装说明 </h2>

````
1.克隆 git@github.com:28468/stytem.git到本地 
2.使用npm install 下载依赖包
3.npm start 启动服务
````
<h2>4.项目结构</h2>

````
├── exam //目录
 ├── mock // 
 ├── dist // 源代码编译后的生成线上环境代码
 ├── node_modules // 依赖的第三方包
 ├── src                // 生产目录
 │   ├── assets         //配置所有接口
 │   ├── components     //各种组件
 |   ├── lang           //语言国际化
 |   ├── models         //命名空间
 |   ├── pages          //静态资源目录
 |   ├── routes         //路由
 |   ├── services       //接口
 |   ├── utils          //视图
 │   ├── index.css      //公共样式
 │   ├── index.js       //封装的方法
 |   ├── router.js      //配置国际化字典
 ├── .babelrc // babel 配置
 ├── .editorconfig // 编辑器 code format 配置
 ├── .eslintrc.js // eslint 配置
 ├── .gitignore // 不纳入 git 版本控制的 /文件夹?/ 列表
 ├── package.json // 项目信息文件
└── README.md // 说明

````
